# NLP

## Regular Expressions

```
Disjunctions: [] or |

e.g. [A-Z], [a-z], [0-9], [A-Za-z0-9]

Negation in Disjunction: ^

e.g. [^Ss]

Others

?: optinal previous char.  e.g. colou?r: color, colour
*: 0 or more of previous char
+: 1 or more of previous char. e.g. [0-9]+
.: any 1 char
$: the end

```
